# sails-app

a [Sails](http://sailsjs.org) application

## Requirements
```
npm install -g grunt-cli
```

```
npm install -g bower
```

### Optional
```
npm install -g sails
```

```
npm install -g nodemon
```

## Run
To have server auto restart when backend scripts are changed
```
nodemon -w api -e js,coffee app.js
```

### Test
```
grunt test
```
